# Tic-Tac-Toe-Game
Tic-tac-toe is a simple, classic two-player game on a 3x3 grid where players alternate placing their mark (X or O) in an empty square, aiming to be the first to get three of their symbols in a row (horizontally, vertically, or diagonally) before the board fills up and results in a draw.
